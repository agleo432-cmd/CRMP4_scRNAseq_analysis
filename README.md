# CRMP4-positive astrocyte scRNA-seq analysis

This repository contains all scripts used for data analysis in:

Shimizu et al., 2026, Astrocytic CRMP4 drives proliferative scar formation and restricts functional recovery after spinal cord injury.

## Data availability
Raw sequencing data are available at GEO: GSE189070.

## Software
- R (v4.5.1)
- Seurat (v5.3.1)

## Reproducibility
All analyses were performed using previously published open-access tools.
Minimal custom code was used.
